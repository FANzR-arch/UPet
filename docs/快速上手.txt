UPet 桌面宠物 —— 快速上手
================================

【安装】
1. 把 UPet.exe 放到任意固定目录（如 D:\Tools\UPet\）
2. 双击运行，宠物出现在桌面右下角，同时弹出宠物画廊
3. 首次运行如被 Windows SmartScreen 拦截：点「更多信息」→「仍要运行」
4. 想开机自动出现：右键宠物 → 设置 → 勾选「开机自启」

【互动】
- 单击宠物：挥手        - 双击：跳跃
- 按住拖动：移动位置（拖动方向会走路）
- Ctrl + 上下拖动 / 滚轮：放大缩小
- 鼠标悬停：它会看你    - 右键：打开菜单

【它还会自己动】
- 待机时随机挥手、张望、思考、散步
- 你连续打字 → 它跟着思考工作
- 你连续点鼠标 → 它做检查动作
- 键鼠 2 分钟没动 → 它打瞌睡；你回来 → 跳起来欢迎
（感知功能只判断"有没有输入"，不记录内容、不联网，可在 设置 里关闭）

【换宠物】
右键 → 选择宠物…：动图画廊里点选，双击卡片直接上桌面
右键 → 获取宠物：
  - Petdex（3600+）/ Awesome Codex Pet：复制页面命令安装，装完画廊里就有
  - CodexPets.net：下载 zip → 获取宠物 → 导入下载的宠物 → 选中 zip
  - Codex 里生成的宠物（~/.codex/pets）自动识别，无需导入

【卸载】
右键 → 退出（先取消开机自启），删除 exe。
配置在 %APPDATA%\UPet\，可一并删除。
