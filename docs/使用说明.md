# UPet 使用说明

UPet 是一个 Windows 桌面宠物小工具：一张宠物精灵图，就能变成常驻桌面、
会互动、会陪你干活的桌宠。**不需要安装 Codex，也不需要任何运行环境。**

---

## 一、下载与安装

1. 下载最新版：**[点这里直接下载 UPet.exe](https://github.com/FANzR-arch/UPet/releases/latest/download/UPet.exe)**
   （或到 [Releases 页面](https://github.com/FANzR-arch/UPet/releases/latest) 选择 zip 包）
2. 把 `UPet.exe` 放到任意你喜欢的目录（比如 `D:\Tools\UPet\`）
3. 双击运行——宠物直接出现在桌面右下角，同时弹出「选择宠物」画廊；
   选完关掉画廊后，宠物会挥手并冒一个气泡把玩法演示一遍（点一下即可收起）
4. 想开机自动出现：右键宠物 → 设置 → 勾选「开机自启」

> **首次运行被拦截？** 看到蓝色的「Windows 已保护你的电脑」，点「更多信息」→「仍要运行」即可。
> 这是因为 exe 没有购买代码签名证书，所有个人开发的小工具都会这样，不是病毒。
> 介意的话可以按 [README](https://github.com/FANzR-arch/UPet#从源码运行--打包) 的说明从源码运行。

卸载：右键宠物 → 退出，然后删掉 exe 即可。配置文件在 `%APPDATA%\UPet\`，想清干净可以一并删除。

## 二、怎么和宠物互动

| 操作 | 效果 |
|------|------|
| 单击宠物 | 挥手 |
| 双击宠物 | 跳跃 |
| 按住拖动 | 移动位置，拖动时会朝拖的方向走路，位置自动记住 |
| Ctrl + 上下拖动 | 连续放大/缩小（25%~300%），大小自动记住 |
| 鼠标滚轮（指着宠物） | 逐级放大/缩小 |
| 鼠标悬停 | 宠物会张望或朝你挥手 |
| 右键 | 打开菜单 |

**它自己也会动**：平时待机呼吸，隔一会儿随机挥手、张望、思考，或在屏幕上散步
（可在 设置 → 自由走动 里关掉）。

**它能感觉到你在干活**（设置 → 感知键鼠活动，默认开启）：

| 你的行为 | 宠物反应 |
|------|------|
| 连续打字 | 进入「思考工作」状态，陪你干活 |
| 连续点击鼠标 | 做「检查」动作 |
| 键鼠 2 分钟没动 | 失落打瞌睡 |
| 打瞌睡后你回来 | 跳起来欢迎你 |

> 隐私说明：感知功能只调用 Windows API 判断「有没有按键/点击」这一个布尔状态，
> 不记录任何按键内容、不存储、不联网。不放心可以随时关闭。

## 三、怎么获得/更换宠物

右键宠物 → **选择宠物…** 打开画廊：每只宠物一张卡片，直接播放动画预览，
点选后按「使用这只」（或直接双击卡片）即可换装。

宠物来源有四种：

### 1. 宠物站下载（最简单）

右键 → **获取宠物**，直达三个宠物站：

| 站点 | 规模 | 用法 | 门槛 |
|------|------|------|------|
| [CodexPets.net](https://codexpets.net/) | 842+ | 点 Download 下载 zip → 右键宠物 → 获取宠物 →「导入下载的宠物」→ 选中 zip，完成 | **不用命令行，推荐先试这个** |
| [Petdex](https://petdex.dev/) | 3600+ | 复制宠物页面的 npx / PowerShell 命令执行，装完就出现在画廊里 | 需要终端，npx 还需要装 [Node.js](https://nodejs.org/) |
| [Awesome Codex Pet](https://awesome-codex-pet.pages.dev/) | 中文社区精选 | 同上，一条命令安装 | 同上 |

### 2. Codex 生成（能做出「你自己」的宠物）

在 Codex 里用官方流程生成宠物后，它会存放在 `C:\Users\<你>\.codex\pets\`。
UPet 会自动扫描这个目录——生成完打开画廊就能看到，无需任何导入操作。

### 3. 导入文件

右键 → 获取宠物 → 「导入下载的宠物（zip/图片）…」，支持：
- 宠物 zip 包（自动从包里挑出精灵图）
- 精灵图本体（透明 WebP/PNG）

### 4. 自己做一张精灵图

格式要求：**8 列 × 9 行（或 11 行）** 的透明 WebP/PNG，每行一个动作，行顺序为：

```
待机 / 向右走 / 向左走 / 挥手 / 跳跃 / 失落 / 等待 / 思考 / 检查（/ 张望A / 张望B）
```

每行帧数不必占满 8 格（按透明度自动检测），格子尺寸按图片大小自动推算
（官方标准是每格 192×208）。做好的宠物欢迎分享到
[本仓库 Issues](https://github.com/FANzR-arch/UPet/issues)。

## 四、设置项速查

右键宠物 → 设置：

| 设置 | 说明 |
|------|------|
| 大小 | 50% ~ 200% 五档预设（Ctrl 拖动/滚轮可无级调节） |
| 速度 | 动画播放节奏：慢 / 正常 / 快 |
| 自由走动 | 关掉后宠物就不会自己在屏幕上散步 |
| 感知键鼠活动 | 打字/点击/挂机反应的总开关 |
| 开机自启 | 勾选后每次开机自动出现 |
| 移到屏幕顶部 / 右下角 | 一键归位 |

所有设置和宠物选择都会自动记住（存在 `%APPDATA%\UPet\config.json`）。

## 五、常见问题

**Q：宠物看起来模糊？**
新版已适配高分屏（125%/150% 缩放），请更新到最新版。

**Q：想同时养两只？**
把 exe 复制一份改名（如 `UPet2.exe`）再运行即可，但两只会共用同一份配置。

**Q：动图能拿去别的地方用吗（微信/PPT/OBS）？**
仓库 `exports/` 目录里有每个动作的独立透明动图（WebP + GIF）；
也可以用 `python scripts/export_animations.py` 从任意精灵图重新导出。

**Q：宠物挡住我点东西了？**
宠物窗口只有它身体那一小块会接收鼠标，透明部分不挡操作；实在碍事拖走就行。

**Q：怎么彻底退出？**
右键宠物 → 退出。如果勾选了开机自启，记得先取消勾选。
